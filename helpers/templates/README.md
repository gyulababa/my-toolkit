# Helpers Templates

This folder contains canonical starter templates for helper-backed schemas.

Templates are:
- static
- human-readable
- intended to be copied and modified

They are NOT used directly by runtime code.

## Zones

Location: `templates/zones/`

Files:
- `zone.json`     – minimal zone object
- `preset.json`  – one preset with zones
- `library.json` – minimal full library document

These templates are compatible with:
- helpers.zones.schema
- helpers.zones.editor
- helpers.zones.serde
